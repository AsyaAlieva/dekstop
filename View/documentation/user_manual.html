Обзор проекта

Это инструкция для использования программы Document Creator.

В программе реализован модуль для генерации отчетов и таблиц для отдела сбыта.

Возможности модуля для отдела сбыта

Этот модуль предназначен для решения 4 задач.

В качестве результата программа выдает по каждой задаче итоговой выходной документ, оформленный в соотвествии со всеми стандартами и ГОСТами.

Ниже описаны все 4 задачи:

  • Составление плана перевозки
  • Составление плана поставки готовой продукции заказчикам
  • Формирование отчета о выполнении плана поставки готовой продукции заказчикам
  • Формирование отчета об остатках на складе
